# HealthyBites
Healthy Bowl is a modern Python-based web app for healthy food delivery and diet planning. It features animated bowl-building, ingredient customization with calories and pricing, meal plans (daily/weekly/monthly), dietician guides, enquiries, and a visually rich, interactive user experience.
